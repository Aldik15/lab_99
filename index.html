Flexbox (гибкий контейнер):

Flexbox - это одномерная система макета, которая работает вдоль одной оси (горизонтальной или вертикальной). Основное назначение Flexbox - упорядочивание и выравнивание элементов вдоль этой оси, обеспечивая легкость и гибкость. Используется для создания сложных макетов внутри контейнера, где элементы могут располагаться в одной строке (горизонтально) или одном столбце (вертикально). С помощью Flexbox можно легко управлять распределением доступного пространства между элементами.

Grid (сетка):

CSS Grid - это двумерная система макета, которая работает как по горизонтали, так и по вертикали одновременно. Основное назначение Grid - создание сложных сеток, где элементы размещаются в ячейках, образуя сетку с определенными строками и столбцами. Используется для создания более сложных и структурированных макетов, таких как таблицы, мультимедийные галереи и многоколоночные макеты. С Grid можно легко определять размеры и расположение элементов внутри сетки, а также управлять промежутками между ними.

Свойства Flexbox

  • display: flex; превращает элемент-контейнер в гибкий контейнер, управляемый Flexbox.
  • flex-direction: определяет направление главной оси контейнера (row, column).
  • justify-content: выравнивает элементы вдоль главной оси контейнера (start, end, center, space-between, space-around).
  • align-items: выравнивает элементы вдоль поперечной оси контейнера (start, end, center, baseline, stretch).
  • flex-grow: указывает, как много свободного пространства должен занять элемент.
  • flex-shrink: указывает, как сильно элемент сжимается, если контейнер уменьшается.
  • flex-basis: устанавливает начальный размер элемента перед распределением свободного пространства.
  • flex: объединяет flex-grow, flex-shrink и flex-basis в одно свойство.
  • align-self: переопределяет выравнивание элемента в контейнере.

Свойства Grid

  • display: grid; превращает элемент-контейнер в сетку, управляемую CSS Grid.
  • grid-template-columns: определяет количество и размеры столбцов в сетке.
  • grid-template-rows: определяет количество и размеры строк в сетке.
  • grid-gap: задает промежутки между ячейками сетки.
  • grid-template-areas: определяет размещение элементов в сетке, используя имена областей.
  • grid-row: определяет строку, в которой размещается элемент.
  • grid-column: определяет столбец, в котором размещается элемент.
  • grid-area: устанавливает ячейку и размещение элемента в сетке.
  • justify-items: выравнивает элементы вдоль горизонтальной оси ячеек (start, end, center, stretch).
  • align-items: выравнивает элементы вдоль вертикальной оси ячеек (start, end, center, stretch).
  • justify-content: выравнивает сетку в контейнере по горизонтальной оси (start, end, center, space-between, space-around).
  • align-content: выравнивает сетку в контейнере по вертикальной оси (start, end, center, space-between, space-around).
